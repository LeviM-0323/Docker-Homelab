# Docker Compose Homelab Files
This repository contains files for hosting and maitainting your own Docker Compose homelab, with instructions to guide you from installing docker-compose, to finalizing your WireGuard configuration, and everything in between.

Please note that there is a lot of content to go over so this repository might be in a permanent state of being unfinished

## Uses
Docker (and docker compose) Have plenty of uses. Just some of the things I've used this homelab setup for are game servers, media servers, app monitoring, and even using download clients or VS code server! There are even docker containers for running entire Linux Installations.

But why use docker, _specifically?_

Well, in my case, it's just what I found out about first. I am aware about things like Kubernetes, Proxmox, TrueNAS and UnRaid, but I never had any interest to me as the Docker containers are already super lightweight and gave me an excuse to learn how to install and use Linux. However, there is nothing wrong with using these other software mentioned above, just note that this repo is meant for Docker Compose **ONLY.**

## Containers Installed
There are about a dozen (maybe 14 exactly at the time of writing) containers currently in use in this homelab, they each have their own unique uses but some of them are sort of complex and rely on each other, so read through a containers purpose before making any changes or removing it

- Radarr
- Lidarr
- Prowlarr
- Qbittorrent
- SabNZBd
- Dashy
- Minecraft Java Server
- TModLoader Server
- Plex
- Uptime Kuma
- Stats
- VS Code Server
- Gluetun
- WatchTower
- WireGuard
